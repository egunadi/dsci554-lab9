Basic Mapbox example

Register with Mapbox and get an access token. The following implements the example from Mapbox Display a map page. To run the code you need to get your own access token and replace YOUR_ACCESS_TOKEN in the code.

<style>
.mapbox_container {
  position: relative;
  height: 300px !important;
  width: 500px !important;
  display: inline-block;
}

#map {
  position: absolute;
  top: 0;
  bottom: 0;
  width: 100%;
}
</style>

<meta name="viewport" content="initial-scale=1,maximum-scale=1,user-scalable=no" />
<script src="https://api.mapbox.com/mapbox-gl-js/v1.12.0/mapbox-gl.js"></script>
<link href="https://api.mapbox.com/mapbox-gl-js/v1.12.0/mapbox-gl.css" rel="stylesheet" />

<div class="mapbox_container">
  <div id="map"></div>
</div>

<script>
  mapboxgl.accessToken = 'YOUR_ACCESS_TOKEN';
  var map = new mapboxgl.Map({
    container: 'map', // container id
    style: 'mapbox://styles/mapbox/streets-v11', // style URL
    center: [-74.5, 40], // starting position [lng, lat]
    zoom: 9 // starting zoom
  });
</script>

Custom Mapbox map

The following is a custom Mapbox map of USC UPC campus using:
  • Dark style (see style URL and Mapbox Dark)
  • A pin for USC (see Add a default marker)

<style>
.mapbox_container {
  position: relative;
  height: 300px !important;
  width: 500px !important;
  display: inline-block;
}

#map2 {
  position: absolute;
  top: 0;
  bottom: 0;
  width: 100%;
}
</style>

<meta name="viewport" content="initial-scale=1,maximum-scale=1,user-scalable=no" />
<script src="https://api.mapbox.com/mapbox-gl-js/v1.12.0/mapbox-gl.js"></script>
<link href="https://api.mapbox.com/mapbox-gl-js/v1.12.0/mapbox-gl.css" rel="stylesheet" />

<div class="mapbox_container">
  <div id="map2"></div>
</div>

<script>
mapboxgl.accessToken = 'YOUR_ACCESS_TOKEN';
var map = new mapboxgl.Map({
  container: 'map2', // container id
  style: 'mapbox://styles/mapbox/dark-v10', // style URL
  center: [-118.285804, 34.021124], // starting position [lng, lat]
  zoom: 14 // starting zoom
});

var marker = new mapboxgl.Marker()
  .setLngLat([-118.285804, 34.021124])
  .addTo(map);
</script>

We are going to use Mapbox to display GeoJSON data for building footprints downloaded from LA GeoHub.

  top: 0;
  bottom: 0;
  width: 100%;
}
</style>

<section class="mapbox_container">
  <div id='map3'></div>
</section>

<script>

var map = new mapboxgl.Map({
  container: 'map3', // container id
  style: 'mapbox://styles/mapbox/dark-v10', // style URL

map.on('load', function () {
