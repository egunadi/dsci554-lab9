Placeholders and areas

Google maps uses KML (Keyhole Markup Language) to to display geographic data. KML is an XML-based file format used to represent geographic features, such as points, lines, polygons, and images, along with their attributes. Google Maps can read and display KML files, allowing users to create custom maps with various data layers. The following is a Google Maps map showing POIs, lines and polygons.
